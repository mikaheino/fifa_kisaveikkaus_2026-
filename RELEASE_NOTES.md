# Release Notes

## v0.1.0 — 2026-05-23

Initial release. Forked from the IIHF MM-kisaveikkaus 2026 codebase and adapted for the FIFA World Cup 2026.

### Features

- Group-stage score predictions for **72 matches** (12 groups × 4 teams).
- **5/3/1 scoring**: 5 p exact, 3 p same goal difference, 1 p correct outcome.
- Full **pudotuspelibracket** veikkaus:
  - Lohkovoittajat ja kakkoset (12 × 2 valintaa)
  - 8 parasta kolmossijaa
  - 16 R16-jatkajaa, 8 puolivälieräjoukkuetta, 4 välieräjoukkuetta
  - 2 finalistia, mestari
  - Maalikuningas (Golden Boot), musta hevonen
- Live countdown timer kohti lukitusta (**11.6.2026 klo 19:00 Helsinki**).
- Leaderboard with breakdown of alkulohko vs bracket points.
- Admin-sivu tulosten syöttöön (group stage + bracket).

### Bootstrap migration

USE ROLE ACCOUNTADMIN;

USE DATABASE STREAMLIT_APPS;
CREATE SCHEMA IF NOT EXISTS FIFA_VEIKKAUS;
USE SCHEMA FIFA_VEIKKAUS;

BEGIN;

-- Schedule (72 group-stage games — populate from official schedule)
CREATE TABLE IF NOT EXISTS FIFA_VEIKKAUS_SCHEDULE (
    ID            NUMBER PRIMARY KEY,
    MATCH_DAY     DATE NOT NULL,
    GROUP_LETTER  VARCHAR(1) NOT NULL,
    MATCH         VARCHAR(200) NOT NULL
);

CREATE TABLE IF NOT EXISTS FIFA_VEIKKAUS_RESULTS (
    ID               NUMBER PRIMARY KEY,
    HOME_TEAM_GOALS  NUMBER,
    AWAY_TEAM_GOALS  NUMBER
);

CREATE OR REPLACE VIEW FIFA_VEIKKAUS_RESULTS_V AS
SELECT
    s.ID, s.MATCH_DAY, s.GROUP_LETTER, s.MATCH,
    r.HOME_TEAM_GOALS, r.AWAY_TEAM_GOALS,
    CASE
        WHEN r.HOME_TEAM_GOALS IS NULL THEN NULL
        WHEN r.HOME_TEAM_GOALS > r.AWAY_TEAM_GOALS THEN 'HOME'
        WHEN r.HOME_TEAM_GOALS < r.AWAY_TEAM_GOALS THEN 'AWAY'
        ELSE 'DRAW'
    END AS WINNER
FROM FIFA_VEIKKAUS_SCHEDULE s
LEFT JOIN FIFA_VEIKKAUS_RESULTS r ON s.ID = r.ID;

CREATE TABLE IF NOT EXISTS FIFA_VEIKKAUS_PREDICTIONS (
    USER_EMAIL       VARCHAR(200) NOT NULL,
    ID               NUMBER NOT NULL,
    MATCH_DAY        DATE,
    MATCH            VARCHAR(200),

CREATE DATABASE ROLE IF NOT EXISTS FIFA_VEIKKAUS_PLAYER;

  ON TABLE FIFA_VEIKKAUS_PREDICTIONS, FIFA_VEIKKAUS_PLAYOFF_PREDICTIONS
  TO DATABASE ROLE FIFA_VEIKKAUS_PLAYER;

  ON ALL TABLES IN SCHEMA FIFA_VEIKKAUS
  TO DATABASE ROLE FIFA_VEIKKAUS_ADMIN;
